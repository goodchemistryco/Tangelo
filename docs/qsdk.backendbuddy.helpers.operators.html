qsdk.backendbuddy.helpers.operators package¶

Submodules¶

qsdk.backendbuddy.helpers.operators.operators module¶

Helper functions on QubitOperators, such as exporting Openfermion QubitOperator to file and vice-versa, or to visualize its coefficients for example.

qsdk.backendbuddy.helpers.operators.operators.ham_of_to_string(of_qb_ham)¶

Converts an Openfermion QubitOperator into a string with information for a Pauli word per line

qsdk.backendbuddy.helpers.operators.operators.print_histogram_coeffs(qb_ham)¶

Convenience function printing a matplotlib histogram of the magnitudes of the coefficient in a QubitOperator object. Combine with the compress method of the QubitOperator class, this allows users to quickly identify what terms in the operator can be discarded, depending on the target accuracy of calculations.

qsdk.backendbuddy.helpers.operators.operators.string_ham_to_of(string_ham)¶

Reverse function of ham_of_to_string : reads a Hamiltonian from a file that uses the Openfermion syntax, loads it into an openfermion QubitOperator
