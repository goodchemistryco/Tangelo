class qsdk.toolboxes.operators.operators.QubitHamiltonian(mapping, up_then_down, *args, **kwargs)¶

Bases: qsdk.toolboxes.operators.operators.QubitOperator

QubitHamiltonian objects are essentially openfermion.QubitOperator objects, with extra attributes. The number of qubit (n_qubits), mapping procedure (mapping), the qubit ordering (up_then_down) are incorporated into the class. In addition to QubitOperator, several checks are done when performing arithmetic operations on QubitHamiltonians.

Attributes:
mapping (string): Mapping procedure for fermionic to qubit encoding

(ex: “JW”, “BK”, etc.).

up_then_down (bool): Whether or not spin ordering is all up then

all down.

Properties:

n_terms (int): Number of terms in this qubit Hamiltonian.
